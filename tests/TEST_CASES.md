# skill-auditor 测试用例

## TC01: G1 大小审计
**输入**: 127 个 skill 目录
**预期**: 输出 >10KB 的 skill 列表 + <1KB 的 STUB 列表
**通过条件**: 超标率和 STUB 率均被统计

## TC02: G2 触发精准度审计
**输入**: 127 个 skill 的 description
**预期**: 统计含 ≥5 触发词的比例 + 含正反例的比例
**通过条件**: 数据非估算，来自逐文件统计

## TC03: G5 失败兜底模板复制检测
**输入**: 多个 skill 的失败兜底表
**预期**: 识别出内容完全相同的兜底表（模板复制）
**通过条件**: 按哈希分组，标记重复组

## TC04: 孤立 skill 检测
**输入**: 127 个 skill 的联动关系
**预期**: 输出无联动标注的 skill 列表
**通过条件**: 孤立率被准确统计

## TC05: 伪审计检测 — grep vs 实际读取
**输入**: TOOL_LEDGER 中文件读取 < 声称审计数
**预期**: 标记为"疑似伪审计"
**通过条件**: 实际读取数 ≥ 声称审计数的 80%
